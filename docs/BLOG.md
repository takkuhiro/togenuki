# はじめに
（全力ポエム。）
リモートワークが当たり前になった時代。
会社員の自分。
上司から催促のメールが飛び交う。相手の顔が見えない。想像するしかない。
「メールが怖い」
受信音が苦手。メールを開くことができない。手が震える。
いざメールを開く。
日本語が読めない。
目で文字を追うことができても頭で理解することができない。
呼吸が荒くなる。会社を休むにしても
そして何もできなくなる。

こうならないためにTogeNukiを提案

（このストーリーはフィクションです）

# 解決する課題

TogeNukiはユーザーに代わって自動でメールを管理します

メールが受信するのが怖い -> 自分の好きな時にTogeNukiを開けば良い
メールを開けない -> 開かなくて良い。TogeNukiが読み込む。
メールを読めない -> 読まなくて良い。音声でアシストする
メールをかけない -> 描かなくて良い。音声で入力し、AIが清書。過去の書き方を参考にそっくりに書く。

この時期に大切なことはとにかく普通の生活を取り戻すことです。
メールという急で過負荷な作業はAIに任せるべきです。

# プロダクト紹介

### 想定シナリオ
1. 対象のユーザーを登録する
    - 学習機能
2. メールを選択して音声で伝える
3. 自動でメールが送信される

### こだわり
- 最小限の労力
精神的に辛い状態というのはやる気が全く起きません。
画面遷移ですら億劫です。
なので、メール作成において画面遷移は行いません。
数回のクリックと音声入力でメールを送信します
万が一メールを確認したい場合は、確認ボタンから確認できます

- 学習
「相手にAIだとバレるかも」という心配もあるかと思います。
そこで、過去の相手とのやり取りを学習します
自分の返信の癖などをAIが深く理解するため、バレる危険性を下げられます

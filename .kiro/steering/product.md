# Product Overview

「TogeNuki」（トゲヌキ）- リモートワークでのメールストレスを軽減するAIコミュニケーションツール

## Core Capabilities

- **メール感情変換**: 冷淡・威圧的なメールを選択したキャラクター口調に変換して読み上げ
- **キャラクター選択**: 複数のキャラクター（全肯定お姉さん、優しい先輩、冷静な執事）から変換スタイルを選択
- **音声入力返信**: ユーザーの口語音声入力をビジネスメールに自動清書
- **コンテキスト学習**: 相手との過去のやり取りパターンを学習し、適切な返信を生成（再学習で最新メールから刷新可能）
- **返信状態同期**: Gmail上で直接返信した場合もTogeNukiが検出し、返信済みステータスを反映

## Target Use Cases

- リモートワーク環境での上司・取引先からのメール受信ストレス軽減
- テキストコミュニケーションでの感情の深読み・誤解の緩和
- ビジネスメール作成の負担軽減（音声→テキスト変換）

## Value Proposition

- **心理的安全性**: メールを直接開かず、好みのキャラクターによるAI読み上げで心象を和らげる
- **コミュニケーション効率**: 口語入力で返信しても、適切なビジネス文体で送信される
- **学習型パーソナライズ**: 相手との過去パターンを踏まえた自然な返信を生成
- **シームレスな返信追跡**: TogeNuki経由でもGmail直接でも、返信状態が一元管理される

## User Journey

1. **設定**: 好みのキャラクターを選択（ボイスプレビュー可能）
2. **受信**: 登録済み連絡先からのメール一覧をダッシュボードで確認
3. **視聴**: 再生ボタンで選択キャラクター口調に変換された音声を聴く
4. **返信**: 音声入力で口語的に返信内容を伝える
5. **送信**: AIが清書したビジネスメールがGmailスレッドに返信として送信される
